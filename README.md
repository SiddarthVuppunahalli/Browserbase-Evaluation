# browserbase-evals

A CLI-first MVP for evaluating browser-agent tasks using Browserbase + Stagehand.

## Setup

1. Run `npm install`
2. Copy `.env.example` to `.env` and fill in your keys.

## Scripts

- `npm run build`: Compile TypeScript to JavaScript.
- `npm run typecheck`: Run TypeScript type checking.
- `npm run run:suite`: Run the evaluation suite.
- `npm run report`: Generate the `findings.md` markdown report.
